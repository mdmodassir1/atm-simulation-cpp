# 🏧 ATM Simulation System (C++)

A console-based ATM simulation system built using C++ that demonstrates core banking functionalities and Object-Oriented Programming (OOP) concepts.

## ✨ Features

- ✅ **User Authentication** - Account number and PIN validation (3 attempts max)
- ✅ **Balance Inquiry** - Check current account balance
- ✅ **Cash Withdrawal** - Withdraw money with validations
- ✅ **User Details Display** - View account holder information
- ✅ **Mobile Number Update** - Change registered mobile number
- ✅ **Mini Statement** - View recent transaction history
- ✅ **Deposit Money** - Add funds to account
- ✅ **PIN Change** - Update ATM PIN securely

## 🛠️ Technologies Used

- **Language:** C++ (C++11/14/17)
- **Paradigm:** Object-Oriented Programming
- **Concepts:** Classes, Encapsulation, Abstraction

## 💻 Installation

```bash
# Compile the code
g++ -o atm atm_simple.cpp

# Run the program
